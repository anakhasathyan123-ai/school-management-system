An Online School Management System developed using **PHP** and **MySQL**.  
This project helps manage students, teachers, subjects, and marks through a simple web interface.
 Features
 - Admin Login
- Add / View Students
- Add / View Teachers
- Add / View Subjects
- Add / View Marks
- MySQL Database Integration
- Simple and clean UI using CSS
Technologies Used
- Frontend: HTML, CSS
- Backend: PHP
- Database: MySQL
- Server: PHP Built-in Server / XAMPP
- Version Control: Git & GitHub
Database setup
1. Open **MySQL Workbench**
2. Create a database:
   ```sql
   CREATE DATABASE school_db;
   1. Open **MySQL Workbench**
3.Import the file:
   scl.sql
4.Update database credentials in:
   config/db.php
